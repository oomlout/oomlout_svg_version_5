# gear_double_stack

**Gears: Double Stack Gear**

Stacks two gear layers (each half the total thickness) with per-layer diameter and extra settings.

## Variables

| Name | Description | Type | Default |
|------|-------------|------|---------|
| diameter | List of two diameters (one per gear layer) in OOBB units. | list | 1 |
| thickness | Total gear stack thickness in mm (split equally). | number | 3 |
| pos | 3-element [x,y,z] position. | list | [0,0,0] |
| extra | Extra variant/modifier string. | string | "" |
| shaft | Shaft type. | string | "m6" |
| holes | Add holes to each layer. | bool | True |
