# jack_basic

**Other: Jack Basic**

Legacy jack component — a plate with an extending cube tab and M6 bolt holes for wall-mounting.

## Variables

| Name | Description | Type | Default |
|------|-------------|------|---------|
| width | Width in OOBB units. | number | 2 |
| height | Height in OOBB units. | number | 2 |
| thickness | Plate thickness in mm. | number | 3 |
