# bearing_plate_hole_shaft

**Bearing Plates: Shaft Hole**

Cuts the central shaft hole in a bearing plate; shape varies by shaft type.

## Variables

| Name | Description | Type | Default |
|------|-------------|------|---------|
| width | Plate width in OOBB units. | number | 1 |
| height | Plate height in OOBB units. | number | 1 |
| thickness | Plate thickness in mm. | number | 3 |
| pos | 3-element [x,y,z] position. | list | [0,0,0] |
| shaft | Shaft type: m6, coupler_flanged, motor_tt_01, motor_servo_standard_01, etc. | string | "m6" |
| extra | Extra variant/modifier string. | string | "" |
| bearing | Bearing type identifier. | string | "608" |

## Category

Bearing Plates
