# bearing_plate_hole_center

**Bearing Plates: Hole Center**

Adds center-zone fastener holes (M3 holes/nuts or circle pattern) matching the bearing type.

## Variables

| Name | Description | Type | Default |
|------|-------------|------|---------|
| shaft | Shaft type identifier. | string | "m6" |
| extra | Extra variant/modifier string. | string | "" |
| thickness | Plate thickness in mm. | number | 3 |
| bearing | Bearing type identifier. | string | "6704" |

## Category

Bearing Plates
