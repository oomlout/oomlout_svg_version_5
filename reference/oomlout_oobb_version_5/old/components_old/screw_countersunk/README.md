# screw_countersunk

**Fasteners: Countersunk Screw**

Generates a countersunk screw with configurable thread size and depth.

## Variables

| Name | Description | Type | Default |
|------|-------------|------|---------|
| radius_name | Screw thread size, e.g. m3, m5. | string | "m3" |
| depth | Screw length/depth in mm. | number | 10 |
