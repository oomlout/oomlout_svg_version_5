# standoff

**Fasteners: Standoff**

Generates a standoff with a through-hole for a selected hardware thread size.

## Variables

| Name | Description | Type | Default |
|------|-------------|------|---------|
| radius_name | Thread size designation, e.g. m3, m5. | string | "m3" |
| depth | Standoff length in mm. | number | 10 |
