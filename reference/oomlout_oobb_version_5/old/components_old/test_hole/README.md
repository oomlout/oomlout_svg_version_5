# test_hole

**Tests: Hole Test Plate**

Plate with a grid of progressively sized hole cutouts for fit testing.

## Variables

| Name | Description | Type | Default |
|------|-------------|------|---------|
| width | Plate width in grid units. | number | 3 |
| height | Plate height in grid units. | number | 3 |
| shaft | Starting hole diameter in mm. | number | 3 |
| bearing | Increment step between holes in mm. | number | 0.5 |
| thickness | Plate thickness in mm. | number | 3 |
| pos | 3-element [x,y,z] position. | list | [0,0,0] |
