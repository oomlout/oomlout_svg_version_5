# bearing_plate_connecting_screw_perimeter

**Bearing Plates: Connecting Screw Perimeter**

Adds countersunk connecting screws at the perimeter of a bearing plate.

## Variables

| Name | Description | Type | Default |
|------|-------------|------|---------|
| thickness | Plate thickness in mm. | number | 3 |
| pos | 3-element [x,y,z] position. | list | [0,0,0] |
| bearing | Bearing type identifier. | string | "6704" |
| extra | Extra variant/modifier string. | string | "" |

## Category

Bearing Plates
