# nut_m3

**Fasteners: M3 Nut**

M3 nut pocket with optional through-hole; wrapper over oobb_nut with radius_name='m3'.

## Variables

| Name | Description | Type | Default |
|------|-------------|------|---------|
| pos | 3-element [x,y,z] position. | list | [0,0,0] |
| type | Geometry type: p/positive or n/negative. | string | p |
| hole | Include a through-hole. | bool | False |
