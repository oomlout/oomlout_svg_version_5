# screw_self_tapping

**Fasteners: Self-Tapping Screw**

Generates a self-tapping screw with configurable thread size and depth.

## Variables

| Name | Description | Type | Default |
|------|-------------|------|---------|
| radius_name | Screw thread size, e.g. m3, m4. | string | "m3" |
| depth | Screw length/depth in mm. | number | 10 |
