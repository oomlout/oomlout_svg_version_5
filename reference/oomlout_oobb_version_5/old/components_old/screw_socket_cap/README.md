# screw_socket_cap

**Fasteners: Socket Cap Screw**

Generates a socket head cap screw with configurable thread size and length.

## Variables

| Name | Description | Type | Default |
|------|-------------|------|---------|
| radius_name | Screw thread size, e.g. m3, m5. | string | "m3" |
| depth | Screw length/depth in mm. | number | 12 |
