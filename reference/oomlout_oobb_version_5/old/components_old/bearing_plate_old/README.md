# bearing_plate_old

**Bearing Plates: Legacy Bearing Plate**

Legacy monolithic bearing plate builder with full hole/shaft/bearing patterns in one call.

## Variables

| Name | Description | Type | Default |
|------|-------------|------|---------|
| shaft | Shaft type identifier. | string | "m6" |
| radius_name | Hole radius name for perimeter holes. | string | "m6" |
| width | Plate width in OOBB units. | number | "" |
| height | Plate height in OOBB units. | number | "" |
| thickness | Plate thickness in mm. | number | "" |
| bearing_type | Bearing type identifier. | string | "608" |
| extra | Extra variant/modifier string. | string | "" |
| pos | 3-element [x,y,z] position. | list | [0,0,0] |

## Category

Bearing Plates
