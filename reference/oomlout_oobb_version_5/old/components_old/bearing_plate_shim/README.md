# bearing_plate_shim

**Bearing Plates: Shim**

Thin shim ring sized to fit between a bearing's OD catch and ID.

## Variables

| Name | Description | Type | Default |
|------|-------------|------|---------|
| bearing_type | Bearing type identifier. | string | "6803" |
| thickness | Shim thickness in mm. | number | 3 |

## Category

Bearing Plates
