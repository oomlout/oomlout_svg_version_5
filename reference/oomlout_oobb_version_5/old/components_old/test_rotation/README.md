# test_rotation

**Tests: Rotation Test Fixtures**

Rotation/transform debug fixtures for item orientation verification.

## Variables

| Name | Description | Type | Default |
|------|-------------|------|---------|
| thickness | Fixture thickness in mm. | number | 3 |
| size | Grid system: oobb or oobe. | string | "oobb" |
| holes | Include reference holes. | bool | True |
| pos | 3-element [x,y,z] position. | list | [0,0,0] |
