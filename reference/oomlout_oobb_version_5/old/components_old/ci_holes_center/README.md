# ci_holes_center

**Circles: Intercardinal Holes Center**

Adds M3 holes and/or slots at the center intercardinal positions of a circular part.

## Variables

| Name | Description | Type | Default |
|------|-------------|------|---------|
| pos | 3-element [x,y,z] position. | list | [0,0,0] |
| slots | Add slot cutouts on the X axis. | bool | True |
| holes | Add hole cutouts on the Y axis. | bool | True |
| inserts | Add threaded insert pockets instead of holes. | bool | False |

## Category

Circles
