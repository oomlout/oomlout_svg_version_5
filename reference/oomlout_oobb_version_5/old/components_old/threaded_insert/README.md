# threaded_insert

**Fasteners: Threaded Insert**

Generates a heat-set threaded insert profile plus pilot hole geometry.

## Variables

| Name | Description | Type | Default |
|------|-------------|------|---------|
| radius_name | Thread size designation, e.g. m3, m4. | string | "m3" |
| style | Insert style profile identifier. | string | "01" |

Fasteners
