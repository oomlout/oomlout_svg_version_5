# bracket_2020_aluminium_extrusion

**Other: 2020 Aluminium Extrusion Bracket**

Bracket plate with M6 and intermediate holes for attaching to 2020 aluminium extrusion.

## Variables

| Name | Description | Type | Default |
|------|-------------|------|---------|
| width | Bracket width in OOBB units. | number | 10 |
| height | Bracket height in OOBB units. | number | 10 |
| thickness | Plate thickness in mm. | number | 3 |
