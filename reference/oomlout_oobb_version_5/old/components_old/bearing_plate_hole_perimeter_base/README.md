# bearing_plate_hole_perimeter_base

**Bearing Plates: Hole Perimeter Base**

Adds standard perimeter fastener holes (miss-middle pattern) to a bearing plate.

## Variables

| Name | Description | Type | Default |
|------|-------------|------|---------|
| width | Plate width in OOBB units. | number | 1 |
| height | Plate height in OOBB units. | number | 1 |
| thickness | Plate thickness in mm. | number | 3 |
| size | OOBB grid size prefix. | string | "oobb" |
| pos | 3-element [x,y,z] position. | list | [0,0,0] |
| shaft | Shaft type identifier. | string | "m6" |
| extra | Extra variant/modifier string. | string | "" |
| bearing | Bearing type identifier. | string | "6704" |

## Category

Bearing Plates
