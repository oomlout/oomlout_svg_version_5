# bearing_wheel

**Other: Bearing Wheel**

Circular wheel with o-ring groove, bearing cutout, and connecting screws.

## Variables

| Name | Description | Type | Default |
|------|-------------|------|---------|
| oring_type | O-ring size identifier (drives wheel diameter). | string | "327" |
| thickness | Wheel thickness in mm (9=single, 15=dual bearing). | number | 9 |
| pos | 3-element [x,y,z] position. | list | [0,0,0] |
| bearing_type | Bearing type for the central cutout. | string | "606" |
