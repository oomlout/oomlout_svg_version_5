# bearing_plate_connecting_screw_center

**Bearing Plates: Connecting Screw Center**

Adds countersunk connecting screws at the center zone of a bearing plate.

## Variables

| Name | Description | Type | Default |
|------|-------------|------|---------|
| thickness | Plate thickness in mm. | number | 3 |
| pos | 3-element [x,y,z] position. | list | [0,0,0] |
| bearing | Bearing type identifier, e.g. 6704. | string | "6704" |
| shaft | Shaft type identifier, e.g. m6. | string | "m6" |

## Category

Bearing Plates
