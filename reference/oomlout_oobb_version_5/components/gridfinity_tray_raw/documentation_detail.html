gridfinity_tray_raw

Gridfinity: Tray Raw

Returns a raw OpenSCAD wrapper around the vendored Gridfinity Extended basic cup, using repo-style public size inputs where gridfinity_width is X, gridfinity_height is Y, and gridfinity_depth is Z.

Category: Gridfinity
Returns: Raw SCAD geometry component dict.
Aliases: None
