# gridfinity_base_tile

**Gridfinity: Base Tile**

Returns the raw OpenSCAD source for a Gridfinity base tile.

## Variables

| Name | Description | Type | Default |
|------|-------------|------|---------|
| distancex | Distance in X direction (mm). | number | 0 |
| distancey | Distance in Y direction (mm). | number | 0 |
| fitx | Fit adjustment in X (mm). | number | 0 |
| fity | Fit adjustment in Y (mm). | number | 0 |
| pos | 3-element [x,y,z] position. | list | [0,0,0] |
| rot | Rotation [rx,ry,rz] in degrees. | list | [0,0,0] |
| type | Geometry type: positive or negative. | string | "positive" |

## Category

Gridfinity
