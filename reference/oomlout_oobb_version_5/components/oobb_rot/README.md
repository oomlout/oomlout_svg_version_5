# oobb_rot

**OOBB Geometry Helpers: Rotation Helper**

Helper that extracts and returns a [rx,ry,rz] rotation list from kwargs.

## Variables

| Name | Description | Type | Default |
|------|-------------|------|---------|
| rot | Explicit [rx,ry,rz] rotation list; returned directly if present. | list | "" |
| rot_x | X rotation in degrees. | number | 0 |
| rot_y | Y rotation in degrees. | number | 0 |
| rot_z | Z rotation in degrees. | number | 0 |

OOBB Geometry Helpers
