# bolt

**Fasteners: Bolt**

Generates a hex-head bolt with specified size and length.

## Variables

| Name | Description | Type | Default |
|------|-------------|------|---------|
| radius_name | Bolt size designation, e.g. m5, m6. | string | "m6" |
| depth | Bolt length in mm. | number | 20 |

Fasteners
