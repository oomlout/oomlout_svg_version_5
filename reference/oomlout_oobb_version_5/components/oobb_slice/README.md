# oobb_slice

**OOBB Geometry Primitives: Slice**

Large cube slice used to clip/intersect geometry.

## Variables

| Name | Description | Type | Default |
|------|-------------|------|---------|
| pos | 3-element [x,y,z] position. | list | [0,0,0] |
| size | [x,y,z] dimensions of the slice cube in mm. | list | [500,500,500] |
| zz | Z anchor: bottom or top. | string | "bottom" |
| mode | Render modes: laser, 3dpr, true. | list | ["laser","3dpr","true"] |

OOBB Geometry Primitives
