oobb_text

OOBB Geometry Primitives: Text

Legacy OOBB text helper that creates centered extruded text with OOBB defaults.

Category: OOBB Geometry Primitives
Returns: List of geometry component dicts.
Aliases: None

Sample Images

`oobb_text` is a compatibility wrapper over the OpenSCAD `text` primitive, matching the legacy helper defaults.
